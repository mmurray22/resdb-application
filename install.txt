1) Install Ninja
--> Need to install cmake, g++ and build-essential
--> Need to install re2c: sudo apt-get install re2c
--> Clone Ninja
--> cd ninja
--> cmake -Bbuild-cmake
--> cmake --build build-cmake
--> Run unit test to see if it passes: ./build-cmake/ninja_test

2) NNG installation
--> Download the latest release of nng. We are using v1.5.2
